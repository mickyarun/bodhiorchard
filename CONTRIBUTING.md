# Contributing to Bodhiorchard

By contributing to this project, you agree to license your contribution under
the Apache License, Version 2.0, and you certify that you have the right to do
so as set out in the Developer Certificate of Origin (https://developercertificate.org/).

All commits must be signed off. Add `-s` to your git commit command:

    git commit -s -m "Your commit message"

This adds a `Signed-off-by: Your Name <your.email@example.com>` line to the
commit message, certifying you have the rights to submit the contribution.
