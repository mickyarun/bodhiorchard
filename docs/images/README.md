# Bodhiorchard screenshots

Drop screenshot PNGs in this directory. The repo README and the public landing
page (`/landing/index.html`) both reference these exact filenames:

| Filename                          | Used by                                            | Suggested size       |
| --------------------------------- | -------------------------------------------------- | -------------------- |
| `dashboard.png`                   | README hero                                        | 2400 × 1350 (16:9)   |
| `screenshot-tree.png`             | README `## Screenshots` and landing-page strip     | 1600 × 1000          |
| `screenshot-bud-board.png`        | README `## Screenshots` and landing-page strip     | 1600 × 1000          |
| `screenshot-slack-triage.png`     | README `## Screenshots` and landing-page strip     | 1600 × 1000          |

Filenames are deliberate — if you change them, also update the references in
`README.md` and `landing/index.html`. PNG is preferred (sharper for UI); JPG is
fine for the hero if file size matters.
